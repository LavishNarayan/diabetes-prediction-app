# 🩺 Diabetes Prediction App

# 📌 Overview

The **Diabetes Prediction App** is a machine learning-based application designed to predict the likelihood of diabetes in individuals based on key health parameters. This project demonstrates the practical implementation of data science and machine learning in the healthcare domain.

---

## 🚀 Features

* Predicts diabetes risk using trained ML models
* User-friendly interface for inputting medical data
* Real-time prediction results
* Clean and interactive UI
* End-to-end pipeline from data preprocessing to deployment

---

## 🧠 Machine Learning Approach

* Data preprocessing and feature scaling
* Model training using classification algorithms
* Performance evaluation using metrics like accuracy, precision, recall, and F1-score
* Model selection based on best performance

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib, Seaborn
* **Frontend:** Streamlit
* **Environment:** Jupyter Notebook

---

## 📊 Input Features

The model takes the following inputs:

* Glucose Level
* Blood Pressure
* BMI
* Insulin Level
* Age
* Skin Thickness
* Diabetes Pedigree Function

---

## 📈 Output

* Displays whether the person is likely to have diabetes or not
* Provides instant prediction based on input values

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/diabetes-prediction-app.git
```

2. Navigate to the project directory:

```
cd diabetes-prediction-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
streamlit run app.py
```
